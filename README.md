# 📌 Project Overview

This project is a **Balloon Shooter Game** developed using **Python and Pygame**.

In this arcade-style game, balloons randomly float upward across the screen, and the player must **burst them using the mouse pointer**.

Each balloon burst increases the score, and balloons continuously respawn with varying speed and movement patterns.

The game demonstrates basic concepts of:

- Game loops
- Collision detection
- Mouse interaction
- Random object movement
- Score tracking

---

# 🎮 Game Features

| Feature | Description |
|------|------|
| Balloon Spawning | Balloons appear randomly on the screen |
| Random Movement | Balloons move in different directions |
| Mouse Shooting | Click balloons to burst them |
| Score System | Tracks number of balloons popped |
| Dynamic Difficulty | Balloons gradually move faster |

# ⚙ Installation

## 🪟 Windows

### 1 Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-balloon-shooter-game.git
```

### 2 Go to project folder

```bash
cd python-balloon-shooter-game
```

### 3 Install required package

```bash
pip install pygame
```

### 4 Run the game

```bash
python Balloon Shooter Game.py
```

# 🧠 How The Game Works

1. Balloons spawn randomly near the bottom of the screen.
2. Each balloon moves upward with a random direction and speed.
3. When the player clicks on a balloon, it bursts.
4. The score increases when balloons are successfully popped.
5. New balloons respawn continuously during gameplay.
